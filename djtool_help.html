DJ Player

DJPlayer is a playlist player tool similiar to VLC that has been optimized for both live and prerecorded shows at KZSU. It includes the following features:
  • Fast song download from YouTube into broadcast ready .wav files
  • Ability to edit and reorder tracks
  • Automatic start/stop of track playback
  • Zookeeper playlist generation
  • Zookeeper live show track spin publishing
  • Accepts dropped files from Finder
  • Generates show MP3 file for uploading
